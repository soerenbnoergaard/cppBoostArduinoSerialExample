# cppBoostArduinoSerialExample

Simple C++ Boost Arduino serial communication. 

## What does it do?

1. Send one command `input` to Arduino/serial device.
2. Receive serial input from Arduino/serial device untill interrupted.

## Requirements

* GCC compiler
* Boost C++ Libraries
